# Score Calculator - WinForms App

A Windows Forms application for managing and calculating scores with statistical results.

## Features
- Add scores to a list
- Clear all data
- Calculate average, maximum and minimum scores
- Input validation
- Separate functions for better code organization

## Technologies
- C#
- Windows Forms
- LINQ for calculations

## Usage
1. Enter a score in the text box
2. Click "Add" to register the score
3. Click "Calculate" to see statistics
4. Use "Clear" to reset all data
